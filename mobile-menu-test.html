Mobile Menu Test Page

This page tests the mobile menu functionality. Click the hamburger menu button in the header to open the mobile menu.

Test Instructions:

  1. Resize your browser window to mobile size (< 768px) or use mobile device
  2. Click the hamburger menu button in the header
  3. Verify the mobile menu slides in from the left
  4. Verify the backdrop appears and is clickable
  5. Verify the close button works
  6. Test on different screen sizes: 320px, 375px, 393px, 768px
